# 🌸 The Pruned Lotus

A cinematic, immersive developer portfolio inspired by the balance of chaos and order — blending elegant design with engineered systems.

---

## ✨ Overview

**The Pruned Lotus** is not just a portfolio — it's an experience.

Built with a fusion of modern frontend technologies and a storytelling approach, this project explores how design, motion, and code can come together to create something memorable.

> Ordered Chaos. Regal Code. Elegant Craft.

---

## 🚀 Features

- 🎬 **Cinematic Hero Section** with animated 3D lotus (Three.js)
- 🌀 **Smooth Scroll & Parallax Effects**
- ✨ **Advanced Animations** using GSAP & Framer Motion
- 🧬 **Text Scramble & Glitch Effects**
- 🌸 **Custom Cursor & Micro-interactions**
- 🧭 **Timeline-Based Navigation**
- 📩 **Stealth Contact System** (Formspree integration)
- 🌌 **Multiverse-Inspired Project Showcase**

---

## 🛠️ Tech Stack

- **Frontend:** React + Vite  
- **Styling:** Tailwind CSS  
- **Animations:** GSAP, Framer Motion  
- **3D Graphics:** Three.js (@react-three/fiber, drei)  
- **Forms:** Formspree  

---

## 📂 Project Structure

- **public/** → Static assets (images, 3D models)  
- **src/components/** → UI + animations (core of your portfolio)  
- **src/hooks/** → Logic abstraction (scroll, effects, etc.)  
- **App.jsx** → Main layout & section composition  
- **main.jsx** → App entry point  
- **configs** → Tailwind + Vite setup

---

## 📬 Contact
